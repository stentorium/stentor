# Function: listisize()

> **listisize**(`items`, `preferredDelimiter?`): `string`

Defined in: packages/stentor-utils/lib/speech.d.ts:15

Builds a speakable and readable list from a set of items.

For example, ["one", "two", "three", "four"] will be transformed
